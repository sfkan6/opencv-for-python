# OpenCV for Python

Некоторые функции OpenCV на Python

Список функций:

* findContours
* threshold (binary, otsu)
* cvtColor (gray, bgr, hsv)
* contourArea
* boundingRect
* morphologyEx (erode, dilate, open, close)
* inRange


Зависимости
* python 3.7+
* [numpy](https://pypi.org/project/numpy/)
